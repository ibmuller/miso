# MISO (Docker)
Docker to run MISO for splicing quantification and analysis

## Available software
- [MISO](http://genes.mit.edu/burgelab/miso/) (run using the `miso` command)
- [samtools](http://htslib.org) 1.5
- [python](https://python.org) 2.7
