# MISO (Docker)
Docker to run splicing analysis using MISO

## Available software
- [MISO](http://genes.mit.edu/burgelab/miso/) (run using the `miso` command)
- [samtools](http://htslib.org) 1.5
- [python 2.7](https://python.org)
